## 全臺36小時天氣預報
* 好友 ID ：@472tymdk<br>
![](https://imgur.com/QMfGYJz.png)
* 公開資料來源：[中央氣象局開放資料平臺](https://opendata.cwb.gov.tw/dist/opendata-swagger.html)
* 使⽤方法及功能說明 ：
  1. 加入好友後自動傳送使用說明。
  2. 使用者輸入「!縣市名稱」即可查詢該縣市36小時內天氣預報，文字間不可含空格，驚嘆號全形、半形皆可。
  <br>例：!臺北市 。
  3. 輸入訊息未以「!」開頭者，判定為無效輸入。
